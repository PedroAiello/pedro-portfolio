LangGraph n8n Supabase Pandas Python

Finflow

Arquiteti um sistema multi-agente de auditoria e reconciliação financeira em Python + LangGraph, modelado como grafo cíclico stateful com 4 rotas de processamento e roteamento condicional.

Implementei entity resolution determinística (rapidfuzz) reconciliando milhares de lançamentos contra base orçamentária e Drive em menos de 1 minuto, com coerção defensiva e duplo backstop anti-loop.

Engenhei resiliência de LLM (Groq/Llama-3.3-70B) com rotação de 6 API keys, exponential backoff e fallback, eliminando falhas por rate-limit. Orquestrei I/O com n8n (Sheets/Drive/e-mail) e Supabase (status real-time), com 43 testes automatizados (contrato + e2e).

  • Resiliência multi-camada
  • 6 API keys + exponential backoff
  • Coerção defensiva + duplo backstop

LangGraph e LangChain Pydantic FastAPI

Assistente Financeiro Autônomo (WhatsApp API)

Desenvolvi um agente conversacional integrado ao WhatsApp (Meta Graph API) para gestão inteligente de despesas, com rotas assíncronas em FastAPI e orquestração stateful via LangGraph e LangChain.

Extraí dados estruturados de áudios e textos via LLMs (ChatGroq) com validação por Pydantic, integrei back-end em Supabase (PostgreSQL) para operações CRUD financeiras e geração de relatórios com tratamento rigoroso de dados temporais (fusos horários).

Implantei a API na nuvem (Render) com monitoramento sintético (UptimeRobot) para evitar cold starts no webhook e garantir alta disponibilidade.

  • Extração de áudio e texto (multimodal)
  • Deploy em nuvem com alta disponibilidade

Engenharia & Inovação

Sou um estudante de Ciência da Computação movido pela obsessão de transformar processos manuais em fluxos de inteligência autônoma. Minha jornada não é definida apenas pelo código que escrevo, mas pelo impacto que ele gera: tempo recuperado, decisões otimizadas e operações escaláveis.

Projeto e construo sistemas multi-agente stateful com LangGraph e LangChain, transformando comandos em linguagem natural em pipelines determinísticos de dados em Python (pandas, FastAPI assíncrona) e automações resilientes em n8n.

Entrego soluções de ponta a ponta combinando back-end robusto em Python e Supabase/PostgreSQL com prototipagem rápida via desenvolvimento assistido por IA (Claude Code e Lovable) — colocando produtos no ar que reconciliam grandes volumes de dados, sobrevivem a falhas de API e eliminam trabalho manual.
